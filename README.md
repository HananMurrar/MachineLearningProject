### Machine Learning Project

Machine Learning (ML) is a field of Artificial Intelligence (AI) that enables computers to learn from data and make predictions or discover patterns without being explicitly programmed.



---

#### Part 1: Supervised Learning – Image Classification
- **Task:** Classifying handwritten digit images
- **Dataset:**  
  [Scikit-learn Digits Dataset](https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_digits.html)

---

#### Part 2: Unsupervised Learning – Text Clustering
- **Task:** Clustering movie reviews based on textual similarity
- **Dataset:**  
  [IMDB Dataset of 50K Movie Reviews](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews/data)

---

#### Part 3: Association Rule Learning – Pattern Discovery
- **Task:** Discovering frequent itemsets and association rules
- **Dataset:**  
  [Groceries Dataset](https://www.kaggle.com/datasets/heeraldedhia/groceries-dataset)
